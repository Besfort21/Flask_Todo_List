# Flask_Todo_List!

## About The Project
Assignment for Web Development course.
To-do list Web application.

### Built With
* Flask
* Flask-SQLAlchemy
* HTML
* CSS
* JavaScript

## Getting Started
1. Download the ZIP.
2. cd to the directory where requirements.txt is located
3. Install dependencies
   ```sh
   pip install -r requirements.txt
   ```
3. Run in your shell
   ```sh
   python app.py
   ```
